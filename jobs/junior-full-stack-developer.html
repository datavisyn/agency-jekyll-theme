Junior full stack web developer

To address the increasing demand for our products, we are looking for a junior developer with experience and/or interest in state-of-the-art web technologies and data visualization. You will enjoy being part of a dynamic team with a flat hierarchy, a great atmosphere, and outstanding employee satisfaction.

Benefits

  • Employment in an aspiring company that is living a start-up character combined with a solid financial base
  • Contributing to innovative visual analysis products for drug development
  • Flat hierarchy with agile management
  • Real work-life-balance and flexible working hours
  • Work remotely for a part of your time

Required technical skills

  • JavaScript (ES 6/next) and/or TypeScript
  • Object oriented programming experience
  • HTML5 + CSS3
  • Source code management systems

Optional technical skills

  • Visualization: d3.js + plotting libraries such as Chart.js
  • Data science: data wrangling using Python, pandas, numpy, R
  • Databases: relational, NoSQL databases, key-value stores, etc.
  • Deployment: Docker, CI, Kubernetes, cloud services (AWS, Azure Cloud), system administration skills, dependency management (webpack, npm, etc.)
  • Design: user experience, graphic design tools
  • Testing: testing strategy, unit tests for web applications, front-end tests (e.g., Selenium)

Soft skills

  • Independent and reliable team player
  • Experience in working on large-scale web-based projects
  • Excellent knowledge of German and English (all code, documents, and documentation are in English, as well as a significant part of internal and external communication)
  • Customer service oriented and prepared to interact with customers and visit customer sites
  • Strong communication skills, prepared to teach workshops and create presentations to customers
In accordance with §9 Abs. 2 GlBG: the minimum monthly salary is € 2461.00 when employed full-time as stated in the collective bargaining agreement. The actual salary is subject to negotiations and depends on experience and qualifications. Apply via email to contact@datavisyn.io. We are looking forward to hearing from you!
